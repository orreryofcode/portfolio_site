Below are a few of the projects that I have worked on.

1. House Marketplace - Mock App

An app that allows users to create a profile and post a listing for a home/apartment for sale or rent. The backend solution uses Google's Firebase as this was something that I was interested in learning at the time. This app has full CRUD support for listings created by users.

Github Repository Live Deployment

2. FoodRush - Mock Website

This is a mock website for a food delivery company. The website design was created in Figma and the actual website was created using HTML, CSS and vanilla JS.

Github Repository

3. Triage - Ticket/Support Desk App

This is a mock web app that was designed with the idea of creating a bug tracker and/or help desk ticket application. The design was created using Figma and the app was built with React and ReduxToolkit. The backend solution was built with MongoDB/Mongoose and also features basic user verification using JSON Web Token and BCrypt.

Github Repository

4. Weather App

A simple web app that allows the user to type in a city name and retrieve the weather. This was built with basic HTML, CSS and JS and uses the OpenWeather API to retrieve weather data.

Github Repository

5. Tekken 7 Iron Fist Companion

A web application for the game Tekken 7 that is designed to provide the community of players with useful information such as frame data, character notes and notable players of each character. This is a passion project that I am currently rebuilding as a lot of the data has changed through recent game updates. I am planning on restyling the entirety of the app as well as I plan to reuse much of the existing structure for Tekken 8 and would prefer it to look nice.

Github Repository
