Below are a few of the projects that I have worked on.

Tekken 7 Iron Fist Companion

A web application for the game Tekken 7 that is designed to provide the community of players with useful information such as frame data, character notes and notable players of each character. This is a passion project that I am building on my own. It is built with React and I am using Tailwind CSS and DaisyUI to assist in some of the styling.

Github Repository

House Marketplace - Mock App

An app that allows users to create a profile and post a listing for a home/apartment for sale or rent. Users can browse listings posted by other users and can edit or delete their created listings. The backend solution uses Google's Firebase as this was something that I was interested in learning at the time.

Github Repository

FoodRush - Mock Website

This is a mock website for a food delivery company. The website design was created in Figma and the actual website was created using HTML, CSS and vanilla JS.

Github Repository

Triage - Ticket/Support Desk App

This is a mock web app that was designed with the idea of creating a bug tracker and/or help desk ticket application. The design was created using Figma and the pages themselves were built using React and Redux-Toolkit. The backend solution was built with MongoDB/Mongoose and also features basic user verification using JSON Web Token and BCrypt.
